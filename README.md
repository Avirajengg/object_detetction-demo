# Real-Time Object Detection with Speech

A browser-based real-time object detection application built using TensorFlow.js and the COCO-SSD model.

## Features

- Real-time webcam object detection
- Bounding boxes around detected objects
- Confidence score display
- Speech synthesis announcing detected objects
- Runs entirely in the browser
- No backend required

## Technologies

- HTML5
- JavaScript
- TensorFlow.js
- COCO-SSD
- Web Speech API
- Canvas API

## How to Run

1. Clone the repository

```bash
git clone https://github.com/yourusername/Real-Time-Object-Detection.git
```

2. Open `object_4.html`

3. Allow camera access

4. Start detecting objects.
